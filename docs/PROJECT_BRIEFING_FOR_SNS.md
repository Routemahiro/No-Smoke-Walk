## 1. プロジェクトの概要

### 何をするサービスか

**NO-SMOKE ALERT Osaka**は、大阪府内での「歩きタバコ」「禁煙エリアでの立ち止まり喫煙」を市民が匿名で報告し、報告データをヒートマップで可視化するWebアプリケーション。

集積されたデータは、行政（大阪府指導員）の巡回ルート最適化に活用することを目指す。

### 解決する課題

- 歩きタバコ・受動喫煙による健康被害（特に子ども・呼吸器疾患者）
- 行政指導員の巡回が非効率（どこに問題があるか把握しにくい）
- 個人が「迷惑だ」と思っても、声を上げる手段がない

### ミッション・ビジョン

| | 内容 |
|---|---|
| **ミッション** | 大阪の街を「煙のない散歩道」に変え、市民の健康と快適な都市環境を守る |
| **ビジョン** | 市民・行政・企業が協働し、迷惑タバコゼロを実現するスマートシティ大阪 |

### バリュー（行動指針）

- **市民主体**: 報告・改善サイクルの中心に市民の声を置く
- **透明性**: データ・プロセス・成果を公開し、信頼を構築する
- **プライバシー尊重**: 匿名報告・データ最小化を徹底し、個人情報を守る
- **協働**: 行政・団体・企業とのオープンな連携を推進する
- **行動喚起**: 小さなアクションが大きな変化を生む体験を提供する
- **継続的改善**: データから学び、UXと施策を絶えずアップデートする

## 2. サービスの機能

### 2-1. 報告フォーム（トップページ）

ユーザーがスマホ・PCから迷惑タバコを報告する機能。

- **位置情報**: GPS自動取得（許可済みなら15〜30秒間隔で自動更新）または手動取得
- **報告カテゴリ**: 「歩きタバコ」「立ち止まり喫煙」の2択
- **送信条件**: 位置情報あり + カテゴリ選択済み
- **レート制限**: 10分間に5件まで（不正投稿防止）
- **プライバシー**: IPアドレスはハッシュ化、個人特定不可、完全匿名

### 2-2. ヒートマップ（/heatmap）

報告データを地図上で可視化する機能。

- **地図エンジン**: MapLibre GL JS（OpenStreetMapタイル）
- **ヒートマップ表示**: 報告が集中するエリアが赤く表示される（白→黄→オレンジ→赤→濃い赤）
- **フィルター**: 表示期間（7/30/90/180日）、カテゴリ（すべて/歩きタバコ/立ち止まり喫煙）
- **ズームイン**: 個別のポイントが表示され、クリックで報告数・カテゴリ・密度の詳細が見られる
- **現在位置表示**: 青い円で自分の位置を確認可能

## 4. サービスの特徴

- **匿名・無料・簡単**: 位置情報ONでカテゴリ選んで送信するだけ（所要時間10秒以下）
- **データの可視化**: 個人の不満を「集合知」としてヒートマップに変換できる
- **行政への提出という明確なゴール**: 単なる愚痴ではなく、報告データの蓄積→行政提出という具体的なアクションに繋がる
- **ヒートマップの視覚的インパクト**: 色のグラデーションで問題エリアが一目で分かる
- **CSVエクスポート**: 蓄積データの分析・ランキング作成などの二次利用が可能
